// 세금 계산 상세
• 연방 소득세: 2024 누진세율 구간 적용. 표준공제 차감 후 과세 (Single $14,600 / MFJ $29,200)
• 401K 공제: 본인 기여분만큼 과세소득 감소. 연간 한도 $23,000 (50세 이상 $30,500) 자동 상한 적용
• Social Security: 6.2% × min(급여, $168,600). 임금 상한 초과분은 SS 면제
• Medicare: 1.45% 전액 + 소득 $200K(Single) 초과분 추가 0.9%
• 주 소득세: 입력한 세율 × (총급여 − 401K 공제)
• 자본이득세: 2024 구간 (0% / 15% / 20%) 과세소득 기준 자동 적용
• NIIT 3.8%: 투자소득이 $200K(Single) / $250K(MFJ) 초과 시 초과분에 추가 부과
• RMD: 73세부터 IRS Uniform Lifetime Table 기준 의무 인출. 인출액은 일반 소득으로 과세
• 401K 인출 (은퇴 후): 부족 생활비만큼 인출. (입력 인출세율 + 은퇴 후 주세) 적용
